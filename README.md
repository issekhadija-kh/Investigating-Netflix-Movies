# Investigating-Netflix-Movies

This analysis focuses on exploratory data analysis on netflix's movie catalog to understand patterns and trends specifically for movies released in the 1990s.

The objective is to analyze the `'netflix_data.csv'` dataset to answer two key questions about movies from the 1990s decade:
                
                1. What was the most frequent movie `'duration'` (min) in the 1990s?
                
                2. How many short action movies are less than 90 minutes were released in the 1990s?
